Better Analytics for Visual Website Optimizer


Firefox: Installation with Greesemonkey plugin
====================================================

1. Install Greesemonkey
2. Open Greesemonkey
3. Click "New User Script" option to open the "New User Script" Form
4. Click "Use Script From Clipboard" button (or just hit OK and paste the script on the next page)
5. IMPORTANT: Enable the script

Chrome: Installation with Tampermonkey plugin
====================================================

1. Install Tampermonkey
2. Open Tampermonkey
3. Click "Add a new script" option to open the "New User Script" Form
4. Paste the Better Analytics script
5. IMPORTANT: In the header of the script, replace @include with @match
6. Save
